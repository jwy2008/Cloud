1. 插件管理使用plug
2. 目前完成了Windows下的部分快捷键(不需要的建议去掉),和wsl和Termux下的剪贴板通信[wsl](https://github.com/equalsraf/win32yank),非wsl用户建议删除,Termux需下载TermuxAPI
